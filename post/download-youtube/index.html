Download Youtube with yt-dlp

January 23, 2023

Goal

Download youtube music as mp3 with ID3 tag information!

Prep

(mac OSX, Python3) ・Install yt-dlp
・Install ffmpeg
・pip3 install mutagen (reference)

Script

Create a script as following, and specify the URL of Youtube you want to download as a parameter $1

$ cat dl-youtube.sh 
#!/bin/bash

IFS=@
read -p "Enter title: " title
read -p "Enter artist: " artist
read -p "Enter album: " album
read -p "Enter filename: " filename

yt-dlp -x -f "ba[ext=webm]" --audio-format m4a $1 -o  "input-youtube.m4a"
ffmpeg -i input-youtube.m4a -ab 256k output.mp3
python3 id3-edit-argparse.py -t ${title}  -a ${artist} -b ${album} -f ${filename}

rm input-youtube.m4a

How it works

The following four tasks are performed in conjunction with each other

  1. drop the file as mp4 with youdube-dl with the best quality & sound quality
  2. convert mp4 to mp3 with audio bitrate 256k using ffmpeg
  3. edit id3 tags
  4. delete intermediate files

id3 tagging

I referred to this article so that I could embed only elements that I might use in my searches. output.mp3 is the temporary name of the file generated by ffmpeg.

https://note.nkmk.me/python-mutagen-mp3-id3/

$ cat id3-edit-argparse.py 
from mutagen.easyid3 import EasyID3
import argparse
import os

parser = argparse.ArgumentParser(description='add id3 tags')
parser.add_argument('-t', '--title', help='title', type=str, required=True) 
parser.add_argument('-a', '--artist', help='artist', type=str, required=True) 
parser.add_argument('-b', '--album', help='album', type=str, default='Youtube') 
parser.add_argument('-f', '--filename', help='filename', type=str, required=True) 

args = parser.parse_args()

path = 'output.mp3'
tags = EasyID3(path)
tags['title'] = args.title
tags['artist'] = args.artist
tags['album'] = args.album
tags.save()

print(tags.pprint())
newfilename = args.filename + '.mp3'
os.rename('output.mp3', newfilename)
