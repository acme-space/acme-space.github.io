Tailwind はおうちクラウドとの相性がよい

January 27, 2023
tags: #tailscale

Tailwind という

ものを使い始めてみた。 OSS の VPN ソフトウェアであり、Linux のカーネルモジュールとしても動作する WireGuard をベースとしたサービスであり、ただでさえ簡単な WireGuard をゼロコンフィグ化したものと言ってよいだろう。

Tailwind はトロントのスタートアップ企業で、techcrunch によると2020年に12億円の資金調達をしている。VPN 界隈のサービスは攻撃や IP 偽装などで利用されているケースが多く正直印象が良くないが、Tailscale はゼロトラストを実現する方法としてエンタープライズ向けのクリーンなイメージが特長かもしれない。

わがやの利用方法

Emby というメディアサーバを使って Spotify にないような音源を聴いてる。 いままでは家のルータで PAT の設定をして自宅のマシンを曝していたが、Tailwind ではその設定が不要となる。体感でしかないが遅延の差はない。

あとは楽譜 PDF ファイルの閲覧などをタブレットでスムーズに行いたいときに SMB クライアントのアプリから直接操作できる点も気に入っている。

その他の利用方法

私は利用していないがメジャーな NAS である QNAP も対応しており、VPN をほぼ意識せずに外出先からも自宅のファイルを閲覧できると いった用途が人気ありそう。 ACL によるアクセス制御もできるがいまは P2P での利用が主で使っていない。NW 単位で利用し始めることがあったら必須の機能となる。 IP の秘匿や Geo Location 変更のために Exit Node として使ったり、他の人とマシン共有する用途があるようだが今の個人でひっそり使うだけで必要性を感じていない。
