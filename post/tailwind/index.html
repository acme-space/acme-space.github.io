Tailwind はおうちクラウドとの相性が抜群

January 27, 2023

Tailwind という

ものを使い始めてみた。 OSS の VPN ソフトウェアであり、Linux のカーネルモジュールとしても動作する WireGuard をベースとしたサービスであり、ただでさえ簡単な WireGuard をゼロコンフィグ化したものと言ってよいだろう。

私は利用していないがメジャーな NAS である QNAP も対応しており、VPN をほぼ意識せずに外出先からも自宅のファイルを閲覧できるといった用途が考えられる。

トロントのスタートアップ企業で、techcrunchによると2020年に12億円の資金調達をしている。VPN 界隈のサービスは攻撃や IP 偽装などで利用されているケースが多く正直印象が良くないが、Tailscale はゼロトラストを実現する方法としてエンタープライズ向けのクリーンなイメージが特長かもしれない。

わがやの利用方法

Emby というメディアサーバを使って Spotify にないような音源を聴いてる。 いままでは家のルータで PAT の設定をして自宅のマシンを曝していたが、Tailwind ではその設定が不要となる。体感でしかないが遅延の差などない。

あとは楽譜 PDF ファイルの閲覧などをタブレットでスムーズに行いたいときに SMB クライアントのアプリから直接操作できる点も気に入っている。

その他の利用方法

IP の Exit として使ったり、他の人とマシン共有する用途があるようだが今の所必要性を感じていない。
