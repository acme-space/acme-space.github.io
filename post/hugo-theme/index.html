Hugo Theme のカスタマイズ

January 21, 2023
tags: #hugo

Hugo Theme

ETCH: https://themes.gohugo.io/themes/etch/

Syntax Highlight の有効化

github を見る限り assets/css 配下に css が用意されていた。 https://github.com/LukasJoswiak/etch/tree/master/assets/css

サンプルコンフィグの highlight = true を利用しても読み込まれていなかったため、 該当するファイルを static/css 配下にコピーした。

mkdir static/css
acme-space% cp themes/etch/assets/css/* static/css/

スタイルを変えたい場合は hugo gen chromastyles --style=vs > ./static/css/syntax.css のようにする。
